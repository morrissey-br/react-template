# react-template

Template básico de projeto React utilizando Vite.

Bibliotecas:
 - TypeScript
 - React
 - React-Router
 - TailwindCSS
 - Storybook
 - Firebase

### Comandos básicos do dev

  - yarn dev - Inicia o servidor de desenvolvimento
  - yarn storybook - Inicia o Storybook
  - yarn build - Gera os arquivos para produção
